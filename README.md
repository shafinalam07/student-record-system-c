# Student Record System (C)

## What it does
This program acts like a digital notebook. You can type in a student's roll number, name, and marks, and the program automatically saves it into a text file named `student.txt`. Even if you close the terminal or turn off your computer, the data stays saved.

## How it works (Simple Terms)
- **Menu:** Gives you a simple 1, 2, 3 choice to add data, view data, or exit.
- **Structures (Struct):** Groups different types of data (text and numbers) together under one student profile.
- **File Handling:** Opens a local text file to permanently write or read the student records.

## How to run it
1. Compile the file using a C compiler: `gcc records.c -o records`
2. Run the program: `./records`
